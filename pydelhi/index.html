About Us

Python Delhi Users Group (PyDelhi) is a volunteer driven organization including students, professionals & entrepreneurs having a common interest in the Python Programming Language.

What We Do

Organize bi-weekly meetups including talks and workshops, that brings together students, developers, experts, researchers, scientists, enthusiasts and companies dealing in the Python programming language.
